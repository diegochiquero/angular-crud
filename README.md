## Angular CRUD

Basic CRUD exercise build in Angular and Bootstrap.

#### Prerequisites

- Node 16.14.2
- Angular 14.1.1
- Bootstrap 5.2.0

#### How do I start?

1. Clone this repository.
2. `npm install`.
3. Go to data folder and run `npm start` to run json-server.  
4. Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.
